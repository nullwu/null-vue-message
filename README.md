# nullwu-vue-message


## 介绍

vue消息提示插件

## 使用


```javascript
npm install -S nullwu-vue-message
import Message    from 'nullwu-vue-message'
Vue.use(Message)
 this.$message({
        type: "success",
        text: "发布成功!"
 });
```
